# check-l10n — Localization Parity Audit

Read-only diagnostic. Never edits files.

## Modules and locales

There are **two modules with localized resources** — do not check only one. The app supports **English and Turkish only**.

| Module   | Locales    | Path                                          |
| -------- | ---------- | --------------------------------------------- |
| `:app`   | `en`, `tr` | `app/src/main/res/values[-tr]/strings.xml`    |
| `:uikit` | `en`, `tr` | `uikit/src/main/res/values[-tr]/strings.xml`  |

If a `values-<xx>` locale directory exists for anything other than `tr` or `night`, flag it as unexpected — the app is EN/TR only.

## Procedure

1. Parse every file above. Extract `<string name="KEY">VALUE</string>`, ignoring entries with `translatable="false"`. Also extract `<plurals>` and `<string-array>` names.
2. For each module, diff EN↔TR key sets and report counts. Show first 25 of each category; note remainder.
3. **Likely-untranslated** (EN value == TR value verbatim, AND value contains a word ≥ 4 letters after stripping `%1$s`-style placeholders and punctuation).
   - Skip these known legitimate duplicates: `app_name`, `app_brand`, `login_header`, `google`, `logo`, `fab_label_pomodoro` (proper nouns / brand names).
4. **Dangling `R.string.*` references** — grep Kotlin under `app/src/main/java` and `uikit/src/main/java` for `R\.string\.([a-z_]+)` and report any key not present in either module's strings.xml.
   - Known false positive: `default_web_client_id` — auto-generated by Google Services Gradle plugin from `google-services.json`. Do not flag.
5. **Unexpected locale dirs** — `ls uikit/src/main/res/` and `ls app/src/main/res/`; flag any `values-*` dir other than `values`, `values-tr`, `values-night`.
6. If everything is clean, say so in one sentence and stop.
7. Otherwise print a summary table, then category details, then offer: "Want me to draft translations for the missing keys? I'll propose values for review."

## Reference script

Use this Python block (stdlib only). The user can run it directly or you can run it via Bash:

```python
import re, glob, os
ROOT = "/Users/beratbaran/AndroidStudioProjects/ToDoApp"
FILES = {
    "app-en":   f"{ROOT}/app/src/main/res/values/strings.xml",
    "app-tr":   f"{ROOT}/app/src/main/res/values-tr/strings.xml",
    "uikit-en": f"{ROOT}/uikit/src/main/res/values/strings.xml",
    "uikit-tr": f"{ROOT}/uikit/src/main/res/values-tr/strings.xml",
}
ALLOWLIST = {"app_name","app_brand","login_header","google","logo","fab_label_pomodoro"}
IGNORED_DANGLING = {"default_web_client_id"}
EXPECTED_LOCALE_DIRS = {"values","values-tr","values-night"}

def parse(p):
    s = {}
    for m in re.finditer(r'<string\s+name="([^"]+)"([^>]*)>(.*?)</string>',
                         open(p).read(), re.DOTALL):
        if 'translatable="false"' in m.group(2): continue
        s[m.group(1)] = m.group(3).strip()
    return s

S = {k: parse(v) for k, v in FILES.items()}
# diffs per module, untranslated, dangling, unexpected dirs — see skill procedure.
```
